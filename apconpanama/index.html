Un Título Va Aquí

Además como casa de software desarrollamos ACI Interfaz de comunicación. A través de este programa usted puede continuar utilizando Peachtree como programa administrativo / contable y emitir el documento fiscal con las funcionalidades que ofrece ACI Modulo de Impresion Fiscal.

Nuestro módulo interfaz es aprobado por la DGI mediante la Resolución 201-9914 del 21 de septiembre de 2011.

Además, tenemos para la disposición de los usuarios de Peachtree, el módulo Formularios Fiscales, el cual le permite generar los informes que solicita la DGI (Formulario 94, Formulario 72, Reporte 43).

Nuestro módulo de formularios fiscales viene con herramientas que le permite gestionar los desembolsos y pagos de caja menuda de una forma más eficiente.

Solicite una cita y con gusto le mostramos las diferentes herramientas que disponemos para un mejor uso de su programa.
